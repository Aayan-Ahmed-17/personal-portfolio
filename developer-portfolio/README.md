## Overview

A premium portfolio template built with **Next.js 16**, **React 19**, and **Tailwind CSS 4**. This version is a **pure frontend implementation**, making it extremely easy to deploy to static hosting services like Vercel or Netlify without needing any backend configuration or environment variables.

---

## Features :sparkles:

- ✅ **Cutting-Edge Tech Stack**: Built with Next.js 16, React 19, and Tailwind CSS 4
- ✅ **Fully Responsive**: Optimized for all devices and screen sizes
- ✅ **SEO Optimized**: Better visibility on search engines with Next.js metadata API
- ✅ **No Backend Required**: 100% frontend logic, no `.env` files or databases needed
- ✅ **Contact Integration**: Simple `mailto:` integration for the contact form
- ✅ **Blog Integration**: Automatically fetch posts from dev.to
- ✅ **Dark Theme**: Beautiful gradient design with Lottie animations
- ✅ **Easy Customization**: Simple data-driven configuration in `utils/data/`

## Installation :arrow_down:

### Prerequisites

- **Node.js**: v18.17.0+
- **npm** or **pnpm**

### Getting Started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/yourusername/personal-portfolio.git
   cd developer-portfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Run the Development Server**
   ```bash
   npm run dev
   ```

---

## Usage :joystick:

### Customize Your Portfolio Data

Manage all content through the files in `utils/data/`:

- `personal-data.js`: Bio, social links, and contact info
- `experience.js`: Job history
- `projects-data.js`: Portfolio projects
- `skills.js`: Technical skills
- `educations.js`: Academic background

---

## Deployment :rocket:

Since this is a pure frontend project, you can deploy it instantly:

### Vercel (Recommended)
1. Push your code to GitHub.
2. Import the project into Vercel.
3. It will auto-detect Next.js and deploy.
